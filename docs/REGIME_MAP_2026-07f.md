# REGIME MAP 2026-07f — карта «актив × стратегия» (два источника)

Эмпирическая карта: какой актив какой ноге подходит, по leave-one-out
вкладу в портфель. Собрана `run_regime_map` на двух источниках.
LOO-дельта: **положительная = балласт** (без актива портфель лучше),
**отрицательная = держатель**. Sharpe gross (масштабо-инвариантен).

Это диагностика для формирования гипотезы. Актив перемещается/
исключается только при совпадении направления на ДВУХ источниках И
наличии механизма (Hurst-тест). Одноисточниковый LOO — не приговор.

## Команды воспроизведения

```bash
# Databento (12 инстр, roll-adjusted continuous)
python -m runners.run_regime_map \
    --trend donchian_vt kama ewmac \
    --mr mr_kelt_confirm mr_keltner mr_lowvol \
    --basket commodity --source databento \
    --panel-dir data/panels/futures --start 2019-01-01 \
    --vt --target-vol 0.20

# yfinance (19 инстр, дневки — второй источник)
python -m runners.run_regime_map \
    --trend donchian_vt kama ewmac \
    --mr mr_kelt_confirm mr_keltner mr_lowvol \
    --basket commodity --source yf --start 2019-01-01 \
    --vt --target-vol 0.20

# Детализация по одной стратегии (solo return/DD/Sharpe + LOO):
python -m runners.run_instrument_contribution --strategy donchian_vt \
    --basket commodity --source databento --panel-dir data/panels/futures \
    --start 2019-01-01 --vt --target-vol 0.20
```

## Что СОВПАЛО на двух источниках (робастные выводы)

- **GC/Gold → ТРЕНД.** Оба источника, самый сильный держатель тренда
  (μLOO −0.18 / −0.11). Ядро трендовой ноги.
- **NG/Natural Gas → MR.** Оба источника (−0.12 / −0.08). Классический
  газовый mean-reversion; в тренде балласт (+0.04 databento).
- **CL/Crude Oil НЕ реверсионный.** Оба: в MR балласт (+0.08 databento,
  +0.29 yf!). Databento даёт тренд-держатель, yf нейтрален — но
  MR-вердикт исключён твёрдо.
- **ZL/Soybean Oil → не MR-стопам.** Databento: балласт во всех трёх
  MR (+0.10/+0.13/+0.24). Держать в тренде.
- **Металлы (SI, PA, PL) тяготеют к MR** на обоих (кроме золота).
- **Соевый комплекс (ZS, ZM, ZW / Soybeans, Meal, Wheat) → MR** на
  обоих (мягко).

- CL тренд-сила: databento −0.09 держатель, yf +0.03 нейтрал. Общее

- HG/Copper: databento слабо-тренд, yf нейтрал/MR.
- Разные корзины (12 vs 19) — прямое сравнение символов ограничено.

## Механизм (зачем это работает)

- **Трендовые (GC, CL):** сильные направленные движения, пробойные
  системы ловят хвост. В MR реверсия к среднему ложно закрывает тренд.
- **Реверсионные (NG, металлы, соя):** range-bound с возвратом к
  среднему; пробой ловит ложные выходы, keltner-реверсия работает.
- **ZL особый:** трендовый по направлению, но MR-стопы режут его
  движения (узкие диапазоны внутри тренда бьют по стопам).

## Следующий шаг

Hurst-аллокатор v2 (variance-ratio Ло-МакКинлея) должен ВОСПРОИЗВЕСТИ
эту карту из свойства инструмента: H(GC)>0.5, H(NG)<0.5, H(CL)>0.5.
Если H-вердикты лягут рядом с LOO-вердиктами — модель валидна и
превращает карту-из-результатов в сайзер-из-свойств. Валидация —
`docs/HURST_VALIDATION.md` (после реализации).

## План сдачи (по активу в неделю)

Чтобы не потерять: каждую неделю фиксируется один актив с его
финальной ногой, механизмом и двухисточниковым подтверждением.
Порядок по убыванию уверенности: GC(тренд) → NG(MR) → CL(тренд) →
ZL(тренд) → соевый комплекс(MR) → металлы(MR) → спорные(HG,PA,PL).
