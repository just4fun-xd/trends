## ПРИОРИТЕТ №1 — Hurst-аллокатор v2 (variance-ratio)

Обоснование: карта актив×стратегия эмпирически показала, что
инструмент надо направлять в ногу по его свойству. NG/PL/ZL/PA —
не-трендовые, в тренд-ноге балласт; NG — лучший в MR. Готовая
валидация модели уже есть.

1.1 Оценщик H через variance-ratio Ло-МакКинлея (statsmodels —
    пользователь снял ограничение numpy/pandas-only). Строже
    агрегированной дисперсии, есть тест значимости отклонения от
    случайного блуждания. VR(q) = Var(q-агрег)/(q·Var(1));
    VR<1 -> реверсия, VR>1 -> тренд.
1.2 Валидация против карты: H(NG), H(PL), H(ZL) должны выйти < 0.5
    (реверсия), H(CL), H(GC), H(ZS) > 0.5 (тренд). Если модель это
    воспроизводит — она работает. Если нет — оценщик плох.
1.3 Аллокатор: вес тренд/MR-ноги на инструмент по его H (непрерывно,
    не переключение). Порог откалибровать по карте, НЕ по результату
    (one-shot).
1.4 Прогон комбо с Hurst-весами против equal-weight donchian+mr_kelt
    через bootstrap разности. Ожидание: Sharpe вырастет за счёт
    выкидывания балласта из каждой ноги.
