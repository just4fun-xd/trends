## ПРИОРИТЕТ 1 — Боевой сырьевой v2 на чистых корзинах (@include)

Собрать финальные ноги из держателей contribution (db 2020–2026):
```
--sleeve donchian_vt:commodity:vt@CL,ZS,SI,ZL,HG,GC \
--sleeve mr_lowvol:commodity:vt@CL,NG,SI,ZW,ZC,GC \
--target-vol 0.60 --parity --rf 0.045
```
Прогнать оба источника. Ожидание: Sharpe >= 1.86 (грязный v2) при
DD < 10%. Это ЧИСЛО ДЛЯ АЛЕКСАНДРА. Затем зафиксировать корзины в
core/config как DONCH_CORE / MRLV_CORE (конфиг, не хардкод команд).
