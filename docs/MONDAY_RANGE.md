# MONDAY RANGE — спецификация стратегии

Weekly opening-range breakout. Бенчмарк Александра: простой недельный
пробой понедельничного диапазона, ~100%/год на крипте интрадей,
переживший крупный крах BTC. Это эталон агрессивной доходности, к
которому мы себя калибруем.

---

## ИДЕЯ (простыми словами)

В понедельник рынок задаёт «коридор» — максимум и минимум первого
периода недели. Дальше всю неделю торгуем ПРОБОЙ этого коридора:
цена вышла вверх — лонг, вниз — шорт (или только лонг на крипте).
Логика: понедельничный диапазон — это «согласие» рынка о справедливой
цене на неделю; выход за него = приход новой информации = импульс,
который часто продолжается.

Это НЕ возврат к среднему (мы не ждём отскок к коридору), а пробой
(ставим на продолжение выхода). По характеру — трендовая стратегия на
недельном горизонте.

---

## МЕХАНИКА (для реализации)

1. **Определение диапазона.** В начале недели (понедельник, или первые
   N часов недели на интрадей) фиксируем:
   - `range_high` = максимум опорного периода;
   - `range_low` = минимум опорного периода.
   Опорный период — параметр: весь понедельник (daily), или первые
   4–24 часа (H1/H4 крипта, торгуется 24/7).

2. **Сигнал пробоя.**
   - close > range_high -> лонг (+1);
   - close < range_low -> шорт (−1) [или 0 для long-only крипты/спота];
   - внутри коридора -> держим текущую позицию или 0.

3. **Сброс.** В начале следующей недели коридор пересчитывается,
   позиция может закрыться/перевернуться.

4. **Стоп/выход (варианты для теста):**
   - противоположная граница коридора как стоп;
   - конец недели (тайм-выход);
   - трейлинг от точки входа.

5. **Сайзинг.** Стандартный VT сверху (как у всех стратегий).
   Осторожно с плечом на крипте (гэпы).

---

## ПАРАМЕТРЫ (сетка для калибровки)

| параметр | варианты | смысл |
|---|---|---|
| опорный период | пн целиком / первые 4h / 12h / 24h | ширина коридора |
| таймфрейм | daily / H4 / H1 | частота |
| направление | long+short / long-only | крипта-спот = long-only |
| выход | граница / конец недели / трейлинг | как фиксировать |
| буфер пробоя | 0 / 0.1% / 0.5% от диапазона | фильтр ложных пробоев |

**One-shot дисциплина:** параметры калибруются на train, тест —
единожды. Не подгонять после теста.

---

## ОЖИДАНИЯ И РИСКИ

- **Ниша:** крипта интрадей (H4/H1), где недельная структура выражена
  и импульсы сильные. На дневном сырье, вероятно, слабее (наш daily-
  тренд уже покрыт donchian).
- **Риск подгонки:** «~100%/год» — это агрессивный бенчмарк на
  конкретном периоде; проверяем на walk-forward, не верим одному году.
- **Риск плеча:** высокий VT на крипте = маржин-колл на гэпе. Держать
  плечо ≤2–3x.
- **Сравнение:** bootstrap против donchian_vt на крипте H4 — Monday
  range должен ПОБИТЬ голый пробой Дончиана, иначе он не нужен
  (Дончиан — тот же пробой, но по скользящему окну).

## ПЛАН РЕАЛИЗАЦИИ

1. Написать `brk_monday_range` в strategies/ (long+short и long-only).
2. Зарегистрировать, прогнать на крипте H4/H1 через ccxt.
3. walk-forward + bootstrap против donchian_vt (крипта H4).
4. Если бьёт — кандидат в крипто-ногу; если нет — закрыть с
   механизмом (Дончиан уже ловит те же пробои эффективнее).
