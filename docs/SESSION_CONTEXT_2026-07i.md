## МЕТОДОЛОГИЧЕСКИЕ ПРИНЦИПЫ (закреплены)

- Sharpe — качество; доходность = Sharpe × вола × плечо (VT).
  «Придушить VT» не улучшает edge (Sharpe масштабо-инвариантен).
- solo_ret инструмента ≠ вклад в портфель (equal-weight делит
  капитал; solo = всё в один актив при плече).
- Балласт — свойство ПАРЫ (актив×стратегия). NG балласт в тренде,
  держатель в MR; ZL наоборот. Корзины ног — из contribution.
- Чемпионство: 2 источника + bootstrap разности (gross). Один
  прогон = гипотеза. Некоррелированный слабый член всё равно портит.
- VT>2-3x = бумажное плечо (бэктест не ловит маржин-колл на гэпе).
