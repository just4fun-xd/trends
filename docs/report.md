## ЧАСТЬ 1. Стратегии в планах (MR семейство)

| Стратегия | Описание | Статус | Ожидаемое применение |
|-----------|----------|--------|-----------------------|
| **mr_atr_stop** | База + жёсткий ATR-стоп на выход: закрытие при `low < entry_price − 2·ATR`, ATR фиксируется на баре входа. Обрезает левый хвост там, где RSI-выход не срабатывает (падение без отскока). | 📋 В планах | Агрессивные возвратники на сырье (Natural Gas, Wheat) и акциях (Tesla, Nvidia), где нужны быстрые выходы при пробое уровня. |
| **mr_time_stop** | База + временной стоп: принудительное закрытие позиции через N баров независимо от P&L. Решает проблему "застрявших" сделок в боковике. | 📋 В планах | Флэтовые активы с низкой волатильностью (Visa, Procter, Gold), где сделка может "висеть" неделями без движения. |
| **mr_scaled** | Масштабируемый вход: размер позиции увеличивается пропорционально отклонению от среднего (пирамидинг в убыток). Требует жесткого лимита на максимальный размер. | 📋 В планах | Высоколиквидные активы с предсказуемыми откатами (S&P 500 фьючерс, Gold, Silver). |
| **mr_ladder** | Лестничный вход: серия ордеров на разных уровнях отклонения (например, −1σ, −2σ, −3σ). Усредняет цену входа, но требует глубокого кармана. | 📋 В планах | Волатильные, но цикличные активы (Natural Gas, Copper, Silver). |
| **mr_trend** | MR-стратегия с трендовым фильтром: вход в возврат к среднему только если глобальный тренд (200 SMA) совпадает с направлением сделки. Отсекает контртрендовые шорты в бычьем рынке. | 📋 В планах | Акции с сильным долгосрочным трендом (Apple, Microsoft, Nvidia) — покупка откатов в бычьем тренде. |
| **mr_lowvol** | База + фильтр волатильности: запрет на вход при всплеске ATR выше медианы. Работает только в "спокойные" периоды. | 📋 В планах | Сырье и акции, где волатильность кластеризуется (Crude Oil, Wheat, Tesla). |
| **mr_keltner** | Возврат к средней линии канала Келтнера (EMA ± ATR). Вход при касании нижней/верхней границы, выход при возврате к EMA. | 📋 В планах | Классические возвратники (Palladium, Platinum, Visa, Mastercard). |
| **mr_connors** | Двухпериодный RSI с пороговыми уровнями 5/95. Покупка при RSI2 < 5, продажа при RSI2 > 95. Очень короткие сделки (1-3 дня). | 📋 В планах | Акции с высокой ликвидностью и частыми микро-откатами (Apple, Tesla, Nvidia). |
| **mr_confirm** | MR с подтверждением: вход только после выполнения дополнительного условия (например, разворот свечного паттерна или пересечение сигнальной линии). Снижает количество сделок, повышает win rate. | 📋 В планах | Активы с шумными сигналами (Crypto, Natural Gas), где нужно фильтровать ложные входы. |
| **mr_short** | Чисто шортовая MR-стратегия: шорт при достижении верхней границы канала, покрытие при возврате к средней. Только для активов с долгосрочным нисходящим трендом или медвежьего рынка. | 📋 В планах | Медвежьи рынки, переоцененные активы (перегретые крипто-альткоины, пузыри). |

**Подходящие OU-стратегии:**
- **ou_asym** — асимметричный OU не шортит тренд, позволяет "оседлать" движение
- **ou_hurst_gate** — фильтр Хёрста определяет трендовый режим и переключает логику

**Физика:** Отрицательная автокорреляция, H < 0.45, движение в канале, цикличность.
**Подходящие OU-стратегии:**

- **ou_volgate** — волатильностный гейт отсекает всплески, оставляет только "нормальный" MR

## ЧАСТЬ 4. Итоговые рекомендации

### 🥇 Топ-3 стратегии для продакшена

1. **ou_asym** — универсальный солдат
   - Commodity: +$406 на $400 (Sharpe 0.70)
   - Equity: +$586 на $700 (Sharpe 0.75)
   - Crypto: +$62 на $300 (Sharpe -0.42) ❌ не работает
   - **Вердикт:** Использовать на сырье и акциях, исключить крипту

2. **ou_jump_asym** — лучший для акций
   - Equity: +$593 на $800 (Sharpe 0.76)
   - Commodity: +$309 на $300 (Sharpe 0.48)
   - **Вердикт:** Топ-1 для equity, топ-2 для commodity

3. **ou_hurst_gate** — лучший для сырья
   - Commodity: +$538 на $700 (Sharpe 0.41)
   - Equity: +$828 на $400 (Sharpe 0.21) — но это за счет Nvidia (+647%)
   - **Вердикт:** Отлично для commodity, осторожно с equity (концентрация)

### ⚠️ Критические замечания

1. **Крипта убивает OU:** Все базовые OU-стратегии теряют 80-100% капитала на крипте. Только модификации (asym, jump_asym, adf) показывают небольшие плюсы.
2. **Концентрация риска:** ou_hurst_gate на equity дает +207% ROI, но 65% прибыли — от Nvidia. Это не диверсификация.
3. **ou_kalman — опасен:** Шарп -0.75 на сырье, -0.59 на акциях, -3.96 на крипте (ccxt 1h). Исключить из продакшена.
